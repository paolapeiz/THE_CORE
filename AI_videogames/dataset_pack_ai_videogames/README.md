# ai_videogames · Dataset Pack PRO · Oxford-IIIT Pet

Este pack no redistribuye las imágenes del dataset.
Incluye utilidades para:
- descargar el dataset con `tensorflow_datasets`
- exportar un subconjunto a carpetas
- crear una narrativa de criaturas RPG

## Archivos
- `download_and_prepare_oxfordpets_tfds.py`
- `export_oxfordpets_subset_to_folders.py`
- `rpg_relabel_example.py`
- `README.md`

## Recomendación docente
Usa `tfds` en clase para evitar líos con archivos.
Usa el export a carpetas solo si quieres trabajar luego con pipelines por directorio.
